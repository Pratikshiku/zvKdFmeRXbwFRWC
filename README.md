## Club Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0751dde1ecf440b4b2cbf8d46b103532/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User (Student), Group Leader, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Activity Information
5. Announcement Information
6. Club Information
7. Personal Center
8. Backend Management


**Group Leader Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Activity Information Management
6. Activity Review Management
7. Announcement Information Management
8. Club Information Management
9. Club Review Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Basic Data Management (Activity Type Management, Announcement Information Type Management, Club Type Management)
5. Activity Information Management
6. Activity Review Management
7. Announcement Information Management
8. Club Information Management
9. Club Review Management
10. Group Leader Management
11. Student Management
12. Carousel Information
